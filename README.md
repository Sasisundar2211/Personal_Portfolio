# **Portfolio — Sasi Sundar**

Static HTML/CSS/JS portfolio presenting applied work in AI/ML, LLM engineering, and backend systems. Content focuses on system design, reasoning workflows, and deployment-oriented engineering rather than visual showcase elements.

## Purpose

Provide a clear, verifiable record of implemented projects, technical competencies, and achievements. Structure emphasizes problem–solution–impact framing to reflect engineering thinking.

## Contents

* End-to-end ML and LLM pipelines
* Agentic reasoning systems and tool-use orchestration
* Retrieval-augmented workflows
* Lightweight model-serving examples
* Technical competencies, education, and assessed achievements
* Minimal contact section with essential information only

## Featured Work

**Procurement Agent System**
LLM-based extraction and consistency scoring for procurement documents.

**Multitool LLM Agent**
API-integrated agent executing controlled tool chains and reasoning steps.

**AI Resume Ranking System**
TF-IDF and ML-based ranking model served through a lightweight API.

## Tech Stack

**Frontend:** HTML, CSS, JavaScript
**Referenced Engineering Tools:** Python, FastAPI, PyTorch, LangChain, scikit-learn
**Hosting:** Compatible with GitHub Pages, Netlify, Vercel, Cloudflare Pages

## Structure

```
/assets          shared media and styles
/css             stylesheets
/js              scripts
/index.html      landing page
/projects/       project summaries
/achievements/   achievements list
/contact/        contact section
```

## Local Use

Clone the repository and open `index.html`.

```
git clone <repo-url>
cd <repo>
open index.html
```

No build process required.

## Deployment

Upload as-is to any static hosting provider. No server-side dependencies.
